# Contexte du monde
L'année 2126. La Grande Catastrophe a ravagé la civilisation il y a 1 siècle.
Les rares survivants se cachent dans les ruines de l'ancien monde, chassant
les créatures mutantes et cherchant les technologies perdues de l'avant-catastrophe.
Chaque jour est une lutte pour la survie. Chaque choix pourrait être votre dernier.

Bienvenue dans le monde post-apocalyptique de Neo-Terra

# Situation initiale
Vous vous réveillez dans un abri abandonné. L'air est lourd, l'odeur de rouille et de
décomposition flotte autour de vous. Le bruit lointain d'une créature vous rappelle que
le danger rôde. Votre première décision pourrait être la dernière...

# Premier choix critique
Devant vous, trois chemins s'offrent dans les ténèbres :
(1) Les égouts - Sûrs mais sombres, peut-être des ressources cachées
(2) Les toits - Exposés mais offrent une vue stratégique sur les menaces  
(3) Les rues - Direct mais infesté, les créatures chassent en meute

Votre choix, %s ? Chaque seconde compte...
